文档抽取结构化问答对功能使用说明
本功能可实现从文档中自动化抽取结构化问答对，整体操作分为文档转文本、文本抽取问答对两个核心步骤，按顺序配置并执行对应命令即可完成操作。
1. 文档转文本
打开doc2text.py文件，在文件末尾配置输入文件地址（待处理的源文档路径）和输出文件地址（转换后文本的保存路径）；
直接执行以下 Python 命令完成文档转文本：
python doc2text.py
2. 文本抽取问答对
打开run.sh脚本文件，在文件中配置好输入地址（上一步转换后的文本文件路径）和输出地址（结构化问答对的保存路径）；
执行以下 Shell 命令完成问答对抽取：
bash run.sh
